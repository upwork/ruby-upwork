    Ruby bindings for Upwork API¶ ↑

    Upwork::Api¶ ↑

    This project provides a set of resources of Upwork API from developers.upwork.com based on OAuth 1.0a.

    Features¶ ↑

    These are the supported API resources:

    • My Info

    • Custom Payments

    • Hiring

    • Job and Freelancer Profile

    • Search Jobs and Freelancers

    • Organization

    • Messages

    • Time and Financial Reporting

    • Metadata

    • Snapshot

    • Team

    • Workd Diary

    • Activities

    Application Integration¶ ↑

    To integrate this library you need to have:

    • Ruby >= 1.9.3

    • OAuth Extension installed (use gem install oauth)

    Installation¶ ↑

    Add this line to your application's Gemfile:

    gem 'upwork-api'

    And then execute:

    $ bundle

    Or install it yourself as:

    $ gem install upwork-api

    Usage¶ ↑

    1. Follow instructions from the Installation section.

    2. Open myapp.rb and type the consumer_key and consumer_secret that you previously got from the API Center. That's all. Run your app as ruby myapp.rb and have fun.'
